# Spam Mail Detection

## Project Overview
This project is a Machine Learning model that classifies emails/messages as Spam or Ham.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Model
- Logistic Regression
- TF-IDF Vectorizer

## Features
- Detects Spam and Ham messages
- User-friendly Streamlit interface
- High prediction accuracy
